## 📖 Project Overview

The AI-Powered Personal Farming Assistant is a smart web application designed to help farmers make better agricultural decisions using data and Artificial Intelligence.

The system analyzes soil nutrients, environmental conditions, and weather data to recommend the most suitable crop. It also provides fertilizer guidance, pest detection support, and useful farming tips to improve productivity.

This project demonstrates how AI and data-driven technology can support modern agriculture.

---

## ✨ Features

- 🌱 AI Crop Recommendation – Suggests the best crop based on soil data  
- 🌤️ Live Weather Information – Displays real-time weather data  
- 🔬 Pest Detection Support – Helps identify crop diseases and pests  
- 🧪 Fertilizer Recommendation – Suggests suitable fertilizers  
- 📊 Farming Dashboard – Displays farming insights and analytics  
- 💡 Smart Farming Tips – Provides useful tips for farmers

## 🤖 Machine Learning Model

- Algorithm: Random Forest Classifier  
- Dataset Size: 2200 samples  
- Number of Crops: 22  
- Model Accuracy: 99.32%

The model predicts the most suitable crop using soil parameters such as:

- Nitrogen (N)  
- Phosphorus (P)  
- Potassium (K)  
- Temperature  
- Humidity  
- Soil pH  
- Rainfall
